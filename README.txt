This application targets android SDK 26, min SDK 26 (Android Oreo 8.0.0)
Usage:
    Most list items can be deleted by long-tapping on them.
    Floating action buttons allow you to add an item to the current list.
    Toolbars may contain extra information and screens.

The Documentation folder contains the user storyboard, APK Deployment screenshots,
and the closing documentation/reflections.
